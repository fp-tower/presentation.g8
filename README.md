A [Giter8][g8] template for FP-Tower presentation using [remark.js][remark.js] and [tut][tut] 

## How to use

```bash
sbt new fp-tower/presentation.g8
```


This template was created from [Rob Norris][tpolecat] presentation on [cofree][cofree]
